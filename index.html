HTML

  • The HTML "html" element represents the root (top-level element) of an HTML document, so it is also referred to as the root element. All other elements must be descendants of this element.
  • The HTML "head" element contains machine-readable information (metadata) about the document, like its title, scripts, and style sheets.
  • The HTML "body" Element represents the content of an HTML document. There can be only one "body" element in a document.

CSS

  • Content/Padding/Boarder/Margin
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS styling can be accessable using : Inline CSS, Internal & external CSS Style Sheet.
  • Ensure that you use the link element from the index.html to the Style sheet.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git -A : Stages the modified or added changes to the pepository.
  • git commit -m "comment about the change" : will add a description of the change made for easier debugging save spots.
  • git clone: retrieve an entire repository from a hosted location via URL
